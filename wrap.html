Flex-Wrap

É uma propriedade que aplica-se a flex contêiners. De acordo com o MDN "a propriedade CSS flex-wrap define se os itens flexíveis são forçados a ficarem na mesma linha ou se podem ser quebradas em várias linhas. Se o argumento for válido, ele define a direção na qual as linhas são empilhadas.", de acordo com o que foi definido em flex-direction. São 3 valores possíveis para flex-wrap: nowrap, wrap, reverse.

Exemplos:

flex-wrap: nowrap

Os itens flexíveis são agrupados em uma única linha, o que pode fazer com que o flex container transborde. O cross-start é equivalente ao início ou antes, dependendo do valor da flex-direction. Este é o valor default.

flex-wrap: wrap

Quando o valor de flex-wrap é wrap, os itens flexíveis são quebrados em múltiplas linhas. A quebra de linha acontece no sentido do cross-axis (eixo transversal). Este é composto por cross-star e cross-end. O cross-start é equivalente ao início ou antes dependendo do valor do flex-direction e cross-end é o oposto do especificado cross-start.
